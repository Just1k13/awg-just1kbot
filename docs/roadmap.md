# Roadmap

## What foundation includes
- Stable project scaffold.
- Basic bot and DB wiring.
- Initial schema for core domain entities.
- Future-proof (but minimal) node/profile linkage.

## Planned phases
1. Single-node kernel backend.
2. Subscription flows.
3. Profile generation/export.
4. Second node support.
5. Anti-abuse controls.
6. Referrals.
7. Hardening and operations.

## Future path highlights
- Start with one kernel backend node.
- Build deterministic subscription state transitions.
- Add profile generation/export pipeline.
- Extend to second node via `ProfileNode` assignment policy.
- Introduce anti-abuse and referral systems only after core reliability.
- Evaluate standalone node-helper later (not in foundation phase).

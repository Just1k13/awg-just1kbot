# just1kbot

Initial project scaffold for `just1kbot`.

## Status

This repository currently contains only the foundational structure required to start implementation.

- Product features are intentionally out of scope for this PR.
- Follow-up work should implement behavior incrementally in small PRs.

## Development

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
pytest
```

## Project layout

- `src/just1kbot/`: package source.
- `tests/`: automated tests.
